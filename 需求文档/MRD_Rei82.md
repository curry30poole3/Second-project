
# 界面

## 主界面


### 界面展示：

1. 主界面正下方显示进度条，进度条和整个播放器同宽。
2. 进度条下方有播放/暂停、上一曲、下一曲，播放模式，四个按钮，居中显示
3. 屏幕中央左侧显示专辑封面（扩展），不能做出来就显示一张贴图，可以让用户自行更换。
4. 屏幕中央右侧显示播放列表



### 功能
#### 核心
1. 音乐播放
2. 音量调节（最好支持鼠标拖动音量大小，或者设置两个按钮中间用数字显示音量百分比，总之尽量得支持鼠标调节）
3. 进度条跟踪（进度条根据歌曲播放进度实时变化，支持拖动跳转歌曲播放进度）
4. 选择指定文件并展示文件夹内的mp3，wav文件
5. 播放模式：单曲循环，列表循环，顺序播放，随机播放
#### 扩展
1. 自定义播放列表
2. 读取歌曲信息，显示歌曲的内嵌的艺术家、标题、专辑时长、码率等信息
3. 读取歌词信息（.lrc文件以及mp3、wav等文件的内嵌歌词），并在软件中显示歌词
4. 支持aac、flac等文件的播放
5. UI有悬浮动效（鼠标停留在对应的按钮上，该按钮的背景会变灰（或者其他颜色，不过得统一））
